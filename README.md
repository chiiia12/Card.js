# Usage

Write anchor with "card" class into div with "box" id.

```html
<div class="cardFrame">
    <div id="box">
        <a href="#" class="card">
            content 1
        </a>
        <a href="#" class="card">
            content 2
        </a>
        <a href="#" class="card">
            content 3
        </a>
        <a href="#" class="card">
            content 4
        </a>
    </div>
  </div>
```

add script tag

```html
<script type="text/javascript" src="js/main.js"></script>
```
